# VulnWeb(Expect on Aug 10) 🔥

A deliberately vulnerable web application built from scratch using the PERN Stack (PostgreSQL, Express.js, React, Node.js) to learn and practice Web Application Security.

Unlike projects that only demonstrate vulnerabilities, VulnWeb is developed as a real web application first and then intentionally introduces security flaws for educational purposes.

---

## 🎯 Project Goals

- Learn secure backend and frontend development.
- Understand how real-world web vulnerabilities occur.
- Practice vulnerability discovery and exploitation.
- Provide hands-on CTF-style security challenges.
- Learn how each vulnerability should be properly fixed.
